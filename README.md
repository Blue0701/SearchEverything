# SearchEverything
## AI-Powered Semantic File Search  

 **A Smarter Way to Search Your Files**  

Tired of searching for files using only names or tags? This project brings **semantic search** to your local files using AI-powered embeddings! Instead of relying on exact filenames, you can find documents, images, and other files based on their meaning.  

##  Key Features (TODO)  
- AI-driven **semantic search** for local files  
- Supports various file types: images, documents, PDFs, images, audio, and more  
- Works on Windows & Linux
